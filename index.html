Lennarb

Lennarb is a lightweight, fast, and modular web framework for Ruby based on Rack. The Lennarb supports Ruby (MRI) 3.0+

Basic Usage

require "lennarb"

Lennarb.new do |router|
  router.get("/hello/:name") do |req, res|
    name = req.params[:name]
    res.html("Hello, #{name}!")
  end
end
